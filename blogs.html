Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element are getElementById, getElementsByClassName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerText property gives the text content of the specific element or node in question. it omits the extra spaces The Inner Content element behaves similarly but istad of only giving the text content of the element or node in target but gives also the text content of its child elements and does not omit extra spaces. As for the innerHTML, it behaves differently. this property return all the html code included inside the selected element.

Question-3: What is event delegation in the DOM?

Event delegation is when we apply event handlers to the parent element of the specific targeted element so that it can fire on all the children elements whenever anyone of them is triggere. this process is facilated by the eventbubbling

Question-4: What is event bubbling in the DOM?

event bubbling is a sort of propagation of events in the dom. when an event is applied on a specific element, it will first of all execute on the target element than will fire on one by one all the parent elements of the target element.
